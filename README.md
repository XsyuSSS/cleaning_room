# 🍳 小廚神 RPG (Personal Edition)

這是一個簡單的互動式網頁遊戲，用於個人廚藝學習的進度追蹤與認領。

## 🚀 功能特點
- **互動劇情**：透過選擇題完成學習任務。
- **技能認領**：點擊「認領」後，該技能會永久儲存在瀏覽器中。
- **進度保存**：使用 LocalStorage 技術，重新整理網頁進度也不會消失。

## 🛠️ 技術使用
- HTML5 / CSS3 / JavaScript (Vanilla)
- 部署於 GitHub Pages

## 📝 使用說明
1. 開啟遊戲網址。
2. 選擇想要練習的項目。
3. 完成後點擊「確認認領」，技能就會出現在下方的技能手冊中。
